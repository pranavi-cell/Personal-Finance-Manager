# Personal Finance Manager

## Description
Personal Finance Manager is a simple web-based application that helps users manage their personal finances by tracking income and expenses. The system allows users to categorize spending, maintain financial records, and analyze spending patterns.

This project demonstrates basic financial data management, expense categorization, and simple analytics using web technologies and a database.

---

## Features
- Add income transactions
- Add expense transactions
- Categorize spending (Food, Travel, Bills, Shopping, etc.)
- Track financial activity
- Generate monthly spending summaries
- Store financial records using a database
